**2. 프로젝트 목적**
---
게임을 통해 자신이 배운 프로그래밍 개념을 실제로 적용하고, 실력을 향상시킬 수 있습니다.

게임을 플레이함으로써 즐거움과 만족감을 느끼게 되고, 그 과정에서 스트레스를 해소하고 새로운 동기부여를 얻을 수 있습니다.
